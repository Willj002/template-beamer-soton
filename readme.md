University of Southampton LaTeX Beamer Template.

This adapts the official presentation style in Powerpoint for LaTeX beamer.

This version has been adapted by me to replicate the current style (Current as of June 2026).

A number of options related to colour and font exist. All of the university colours have been added, so the presentation colours can be swapped as well. Moreover, the Georgia font can be used to add emphasis on content such as quotations.
